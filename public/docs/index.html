API Documentation

API v2 hla endpoint

  • Returns all sample phenotypes and hla gene loci
  • Returns data for valid loci names and no data for invalid loci names
  • Returns only data for selected criteria and returns genetic data for only specified loci
  • Returns only data that meets all search criteria
  • Returns sample pheonotypes and gl strings for selected genes

API v2 kir endpoint

  • Returns all sample phenotypes and kir gene loci
  • Returns data for valid loci names and no data for invalid loci names
  • Returns only data for selected criteria and returns genetic data for only specified loci
  • Returns only data that meets all search criteria
  • Returns sample pheonotypes and gl strings for selected genes

API v2 sample endpoint

  • Filter API results by sex
  • Filtering by disease code
  • Filtering results by multiple parameters
  • Include only controls in the response
  • Querying dataset with valid INDIGO credentials
  • Superusers filtering by sample source
  • Superusers receive data for any/all institutions
  • Users only receive data for their own institution's samples
  • Users should only get controls for their own institution

Credentials

  • Redirect to sign up on request with bad credentials
  • Redirect to sign up on request with missing credentials
